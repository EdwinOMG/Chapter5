Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

To use the sticky position, simply apply the position: sticky; property to the desired element. Its position will be relative to the nearest ancestor with a scrolling mechanism.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

The top and left properties were set to 90 pixels and 30 pixels respectively. This shifted the relative box 90 pixels down and 30 pixels to the right from its initial position.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

The top and right properties were set to 90 pixels and 10 pixels respectively. This positioned the fixed box 90 pixels down and 10 pixels from the right edge of the viewport.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

The bottom and right properties were set to 0 pixels each. This positioned the absolute box to the bottom-right corner of its containing parent element.
